3.2 Project Schedule with Milestones & Schedule Margin

Milestone M1: System Requirements Definition. Target Month: Month 1. Deliverable: Requirements document aligned with NASA HRP health monitoring needs. Schedule Margin: 10%.

Milestone M2: Microfluidic Chip CAD Design. Target Month: Month 2. Deliverable: CAD design and fluidic simulation results. Schedule Margin: 10%.

Milestone M3: Sensor Electrode Prototype. Target Month: Month 3. Deliverable: Initial biosensor test results. Schedule Margin: 10%.

Milestone M4: Phase 1 Decision Gate. Target Month: Month 4. Deliverable: Design review and approval for fabrication. Schedule Margin: 15%.

Milestone M5: ML Model Development. Target Month: Month 6. Deliverable: Preliminary biomarker classification model. Schedule Margin: 10%.

Milestone M6: System Integration Prototype. Target Month: Month 8. Deliverable: Integrated microfluidic + ML prototype. Schedule Margin: 10%.

Milestone M7: Functional Testing & Validation. Target Month: Month 10. Deliverable: Lab validation data and performance analysis. Schedule Margin: 15%.

Milestone M8: Final Prototype Demonstration. Target Month: Month 11. Deliverable: Prototype demonstration with simulated urine samples. Schedule Margin: 10%.

Milestone M9: Final Report & Documentation. Target Month: Month 12. Deliverable: Technical report, design documentation, and NTR submission. Schedule Margin: 10%.

The schedule includes ~10–15% margin per major milestone to mitigate delays associated with fabrication, component procurement, or system integration.
